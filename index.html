How to use git

Create new Repository

create an new Repository by clicking on "+" symbol
add Repository name and title.

Initialize Repository

Open git bash in folder where you want to initialize git
type: git init
it will initilze git.

Configure Name and email

cofigure your name and email if using git for first time type:
git --global user.name yourName
git --global user.email yourEmail

Adding Files to Staging Area

Add files to local Repository staging area by typing:
git add filename
or to add all files at once:
git add .

Commiting Changes on Local Repository

Commit changes to local Repository by typing:
git commit -m "your commit message here."

Pushing Files on Remote Repository

Connect to Remote Repository by typing:
git remote add origin https://github.com/atiagull/how-to-use-Git.git
Push files on Remote Repository by typing:
git push -u origin

Creating New Branch

Connect to Remote Repository by typing:
git checkout -b newbranch
